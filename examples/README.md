# Examples

Reference AI Parlance (`.aip`) specs for learning, validation, and transpiler tests.

| File | Description |
|---|---|
| [minimal.aip](minimal.aip) | Smallest valid v0.1 spec (`app` + one entity + CRUD) |
| [crm-reference.aip](crm-reference.aip) | CRM with users, leads, tasks, policies, API, and workflows |

Documented at [docs.aiparlance.org](https://docs.aiparlance.org) · [source on GitHub](https://github.com/eudameron/aiparlance/tree/main/examples).
